About This Project

Static Website Hosting using Amazon S3 with GitHub

Project Objective

The main objective of this project is to host a static website on Amazon S3 and understand AWS cloud deployment. This project also demonstrates how GitHub is used to manage website source code.

Problem Statement

Many beginners find cloud deployment difficult. This project provides a simple and cost-effective way to host a website using Amazon S3 without managing servers.

Project Features

  • ✔ Static Website Hosting
  • ✔ Public Website Access
  • ✔ Beginner Friendly
  • ✔ GitHub Integration
  • ✔ Responsive Design
  • ✔ Cloud Deployment

Advantages

  • ✔ High Availability
  • ✔ Secure AWS Infrastructure
