Architecture Explanation

1. Website files are developed using HTML and CSS in VS Code.

2. The source code is uploaded to GitHub for version control.

3. The website files are uploaded to an Amazon S3 bucket.

4. Static Website Hosting is enabled in Amazon S3.

5. The website becomes publicly accessible through the S3 Website Endpoint.

Project Flow

  • Develop Website
  • Upload Files to GitHub
  • Create Amazon S3 Bucket
  • Upload Website Files
  • Enable Static Website Hosting
  • Configure Bucket Policy
  • Access Live Website
